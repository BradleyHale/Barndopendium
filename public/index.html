- hello, and welcome to Barndopendium! here's how this program works: for everything you want to search by, click the checkbox and input the value you want to search for.

- for any values involving measurements (length, width, sidewallLength), input the value you want to search by IN FEET

NOTE: when searching by more than one thing at a time, the program will display results that meet EITHER criteria.

so if you're searching for plans that have say, 3 bedrooms and 2 bathrooms, this program will display all results

that have 3 bedrooms, OR 2 bathrooms. so you may have results that meet one criteria but not the other.
